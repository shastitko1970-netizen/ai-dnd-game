# Changelog

## [1.0.0] - 2025-12-13

### Initial Release

**Frontend**
- ✅ Next.js 14 app with App Router
- ✅ React 18 with TypeScript
- ✅ Zustand state management
- ✅ Tailwind CSS styling
- ✅ Home page with features overview
- ✅ World selection page
- ✅ 3-step character creation wizard
- ✅ Game session page with narrative and actions
- ✅ Custom content management (races, classes, feats)
- ✅ Full API client with axios

**Backend**
- ✅ Fastify HTTP server
- ✅ TypeScript with strict mode
- ✅ RulesEngine for D&D 5e mechanics
- ✅ CustomContentManager for user-created content
- ✅ GameManager for session management
- ✅ CharacterService for character creation
- ✅ API routes for all operations
- ✅ JSON-based storage (MVP)
- ✅ CORS enabled
- ✅ Error handling middleware

**Features**
- ✅ Full D&D 5e rule system
- ✅ Character creation with 6 official races and 6 classes
- ✅ Custom content creation (races, classes, feats)
- ✅ Content validation
- ✅ Core + custom content merging
- ✅ Game session management
- ✅ Ability score calculations
- ✅ AC and HP calculations
- ✅ Dice rolling mechanics

**Documentation**
- ✅ Complete README
- ✅ Installation guide
- ✅ Quick start guide
- ✅ Development guide
- ✅ API documentation
- ✅ Project structure guide
- ✅ Changelog
